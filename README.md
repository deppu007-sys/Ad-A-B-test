# Ad A/B Test Analyzer (Python)

## 📌 Overview
This project compares two ads and determines which performs better using CTR (Click Through Rate).

## 🚀 Features
- Calculates CTR for both ads
- Compares performance
- Beginner-friendly
- Works on Android (Pydroid 3)

## 🧠 Formula
CTR = Clicks / Impressions

## ▶️ How to Run
1. Open in Pydroid 3
2. Run ab_test_analyzer.py
3. Enter ad data

## 📊 Example
Ad A CTR: 0.05  
Ad B CTR: 0.058  
Winner: Ad B
